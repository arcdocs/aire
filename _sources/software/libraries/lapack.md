# LAPACK

LAPACK (Linear Algebra PACKage) is an open-source Fortran library for linear algebra. Please see the [official documentation](https://www.netlib.org/lapack/) for details on usage.

## The LAPACK module on Aire

| Version |  Command                    |
|---------|-----------------------------|
| 3.12.0  | `module load lapack/3.12.0` |
