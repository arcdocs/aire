# File and Data Management

Efficient file and data management is essential for productive work on Aire. This section provides an overview of key topics related to handling data on the HPC system:

- **Best Practice**: Learn strategies for organising, storing, and managing your data effectively.
- **Storage Options**: Understand the available storage locations and their specific purposes.
- **Filesystem Quotas**: Monitor and manage your allocated storage limits to avoid disruptions.
- **Data Transfer**: Explore reliable methods for transferring data to and from Aire.
- **Data Sharing**: Discover ways to collaborate and share data securely with other users.
